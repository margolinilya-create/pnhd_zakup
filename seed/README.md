# seed/

Заглушка под сид-данные. Реальные данные (24 ткани, 2 поставщика, 36 рёбер, 42 SKU)
въедут в **Фазе 2** через `backend/prisma/seed.ts` — источник истины для значений:
`claude_code_playbook.md`, ПРОМТ 2.

Несматченные SKU (тканей нет в базе — пользователь добавит через UI) фиксируются в
`seed/UNMATCHED.md` на той же фазе.
